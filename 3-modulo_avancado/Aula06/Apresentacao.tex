\section{YOLO - You Only Look Once}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\begin{frame}{YOLO - You Only Look Once}
\begin{itemize}
\item {YOLO - You Only Look Once} é uma rede neural {\it end-to-end} que utiliza uma estratégia diferentes dos demais algoritmos.
\item Divide uma imagem em um {\it grid} $S \times S$, e cada célula do gir pode predizer 2 bounding boxes
\item Utiliza uma rede separada para predizer as {\it Region Proposals}
\end{itemize}

\vspace{1cm}
\tiny{Paper: \href{https://machinelearningmastery.com/object-recognition-with-deep-learning/}{A Gentle Introduction to Object Recognition With Deep Learning}}
\end{frame}
