\begin{frame}{GoogLeNet}{Inception v3}

\begin{itemize}
\item {\bf Inception v3} é uma variante do {\it Inception v2} que adiciona BN-auxiliar.
\item BN auxiliar refere-se à versão em que a camada totalmente conectada do classificador auxiliar também é normalizada, não apenas convoluções.
\item Portanto, o modelo {\it Inception-v3} trata-se do [Inception-v2 + BN auxiliar].
\end{itemize}

\section{Vanishing Gradient Problem}
\begin{frame}{Vanishing Gradient Problem}
\begin{itemize}
\item Quanto mais camadas se adiciona em uma rede, menor ficam os gradientes utilizados para atualização dos pesos durante a etapa de {\it back propagation}.
\item Isso dificulta o treinamento de redes neurais muito profundas, contrário ao que se imaginava.
\item Esse problema é chamado de {\bf Vanishing Gradient Problem}
\end{itemize}

\end{frame}

\section{ResNet}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\begin{frame}{ResNet}{Problema}
Empilhar convoluções $3 \times 3$ funcionava bem até 20/30 camadas.

\begin{figure}[h]
\includegraphics[width=12cm]{imgs/test_error_number_layers.png}
\end{figure}

\vspace{1mm}
\tiny{Paper: \href{https://arxiv.org/pdf/1512.03385.pdf}{Deep Residual Learning for Image Recognition}}

\end{frame}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\begin{frame}{ResNet}
\begin{itemize}
\item O modelo {\bf ResNet} propõe uma solução muito simples: introduzir um ``atalho'' entre as camadas.
\item Dessa forma, os erros de treinamento não aumentarão, pois o que foi aprendido nas camadas anteriores não é perdido.
\item E como a informação `flui'' diretamente pelo atalho, evita o {\it Vanishing Gradient Problem}.
\end{itemize}


\vspace{1cm}
\tiny{Paper: \href{https://arxiv.org/pdf/1512.03385.pdf}{Deep Residual Learning for Image Recognition}}
\end{frame}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\begin{frame}{ResNet}

\section{DenseNet}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\begin{frame}{DenseNet}
\begin{itemize}
\item O modelo {\bf DenseNet} é muito similar ao ResNet, com algumas diferenças fundamentais.
\item Utiliza uma estratégia aditiva, que combina a saída da camada anterior com a saída da próxima camada.
\item As saídas das camadas são concatenadas, aumentando a profundidade da informação extraída.
\end{itemize}

\vspace{1cm}
\tiny{Paper: \href{https://arxiv.org/pdf/1608.06993.pdf}{Densely Connected Convolutional Networks}}

\end{frame}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\begin{frame}{DenseNet}
Nas CNNs tradicionais, a imagem de entrada passa por múltiplas convoluções e obtém {\it features} de alto nível.
\begin{figure}[h]
\includegraphics[width=10cm]{imgs/densenet_block.png}
\end{figure}

\vspace{2cm}
\tiny{Fonte: \href{https://towardsdatascience.com/review-densenet-image-classification-b6631a8ef803}{Review: DenseNet - Dense Convolutional Network }}

\end{frame}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\begin{frame}{DenseNet}

Nas {\bf ResNet} utiliza-se soma elemento-a-elemento ({\it element-wise}), o que faz com que a informação flua de um módulo para outro.
\begin{figure}[h]
\includegraphics[width=10cm]{imgs/densenet_block1.png}
\end{figure}

\vspace{2cm}
\tiny{Fonte: \href{https://towardsdatascience.com/review-densenet-image-classification-b6631a8ef803}{Review: DenseNet - Dense Convolutional Network }}

\end{frame}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\begin{frame}{DenseNet}

Na {\bf DenseNet}, a saída de cada camada é conectada com todas as camadas posteriores, utilizando concatenação.

\begin{figure}[h]
\includegraphics[width=10cm]{imgs/densenet_block2.png}
\end{figure}

Dessa forma, cada camada recebe um ``conhecimento coletivo'' de todas as camadas precedentes.

\vspace{2mm}
\tiny{Fonte: \href{https://towardsdatascience.com/review-densenet-image-classification-b6631a8ef803}{Review: DenseNet - Dense Convolutional Network }}

\end{frame}
